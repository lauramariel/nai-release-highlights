  - title: "Support for New Models"

    category: "Platform"
  - title: "Updated Minimum Software Requirements"
    description: "Updated minimum software requirements for deploying Nutanix Enterprise AI 2.1, including new version floors for Kubernetes, Cert-manager, Kserve, Istio, Knative-serving, NVIDIA GPU Operator, Kube-prometheus-stack, Nutanix CSI Driver, Nutanix Files, and Nutanix Objects."
    category: "Platform"
  - title: "Support for OpenAI Compatible Clients"
    description: "Endpoints created in Nutanix Enterprise AI can now be accessed using any OpenAI-compatible client."
    category: "API"
  - title: "Enhancements to the Pulse Payload"
    description: "Nutanix Enterprise AI now shares its application version number with Nutanix through Pulse telemetry."
    category: "Platform"
  - title: "Support for Taints and Tolerations"
    description: "Users can now taint Kubernetes nodes and enable tolerations on pods so that only pods with matching tolerations are scheduled on tainted nodes, preventing non-GPU workloads from being scheduled on GPU worker nodes."

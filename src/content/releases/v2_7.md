version: "2.7"

summary: "Nutanix Enterprise AI 2.7 introduces General Availability of the AI Gateway, local MCP server deployment, batch inferencing, enhanced model and endpoint security via Palo Alto Networks Prisma AIRS, unified endpoints with external AI providers, and an agent application in NAI Labs."
features:
  - title: "General Availability of AI Gateway Mode"
    description: "AI Gateway mode is now generally available (GA) in Nutanix Enterprise AI. It provides cost control and governed access to models and tools. AI Gateway Mode is automatically enabled on fresh installs of NAI 2.7 and on upgrades from NAI 2.6 standard mode. Supported AI providers include Anthropic, AWS Bedrock, Azure Open AI, Cohere, GCP Vertex AI, Google Gemini, Mistral, and Open AI."
    category: "Platform"
  - title: "Support for Local MCP Server Deployment and Configuration"
    description: "NAI now supports the local deployment of SSE transport-enabled MCP servers within the NAI cluster and manages access to their tools. MCP Connectors support specification version 2025-06-18 and the following APIs: tools/*, prompts/*, and resources/*."
    category: "Platform"
  - title: "Batch Inference Support"
    description: "NAI introduces batch inferencing to support the efficient processing of large-volume inference requests, such as vector embeddings, translations, and summarizations."
    category: "MLOps"
  - title: "Enhanced Security for Models and Endpoints"
    description: "NAI now integrates with Palo Alto Networks Prisma AIRS to provide inline model scanning and endpoint security, available to users with the ML Admin role. Capabilities include configuring security scanning with Prisma AIRS credentials, automatic model security status monitoring with detailed scan results for Hugging Face models, on-demand re-triggering of scans, endpoint security scans for Local Endpoints, a unified security dashboard for models and endpoints, and credential lifecycle management."
    category: "Security"
  - title: "Support for Creating Unified Endpoints with External Providers"
    description: "NAI now enables the creation of Unified Endpoints backed by external AI providers, available to users with the ML Admin role. Supported providers are Anthropic, AWS Bedrock, Azure Open AI, Cohere, GCP Vertex AI, Google Gemini, Mistral, and Open AI."
    category: "Platform"
  - title: "Agent Application in NAI Labs"
    description: "A new Agent application in NAI Labs connects a language model endpoint and MCP tools for agent interactions, allowing users to submit prompts and receive results from the configured agent."
    category: "MLOps"
  - title: "Support for New Models"
    description: "New LLMs are now supported: Hugging Face models google/gemma-4-E2B-it, google/gemma-4-26B-A4B-it, google/gemma-4-31B-it, nvidia/NVIDIA-Nemotron-3-Nano-30B-A3B-BF16, and nvidia/NVIDIA-Nemotron-3-Nano-30B-A3B-FP8; and NVIDIA NIM model Llama Nemotron VL Embed 1B."
    category: "MLOps"

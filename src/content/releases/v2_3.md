version: "2.3"
release_date: 2025-05-07
summary: "Nutanix Enterprise AI 2.3 introduces CPU inferencing with Intel AMX, vision model support, endpoint hibernation, and a new licensing mechanism. This release also adds Japanese language support, OpenTelemetry metrics export, self-managed PostgreSQL database installation, and several new AI models from Hugging Face and NVIDIA."
features:
  - title: "CPU Inferencing Using Intel AMX"
    description: "Nutanix Enterprise AI now supports Intel Advanced Matrix Extensions (Intel AMX), a built-in accelerator that improves the performance of deep-learning training and CPU-based inferencing."
    category: "Performance"
  - title: "Support for Vision Models"
    description: "Nutanix Enterprise AI introduces support for vision-type LLMs that can process and understand both text and visual inputs, such as images or videos."
    category: "Core"
  - title: "Japanese Language in User Interface"
    description: "Users can now change the language settings in Nutanix Enterprise AI to Japanese, adding Japanese as a supported UI language."
    category: "Platform"
  - title: "Support for Importing Hugging Face Models Using Model URL"
    description: "Users can now import an LLM not validated by Nutanix directly from Hugging Face by providing the model URL, avoiding the need to download and store the model locally and making it efficient for environments with limited storage space."
    category: "MLOps"
  - title: "Exporting Metrics Using OpenTelemetry Collector"
    description: "Users can configure OpenTelemetry Collector to export Nutanix Enterprise AI metrics and view them in any OpenTelemetry-compliant observability tool."
    category: "Platform"
  - title: "Nutanix Enterprise AI Licensing"
    description: "A new licensing mechanism is introduced to manage product access and usage. The licensing framework supports on-premises, hybrid, and cloud deployment scenarios, and all deployments now require a valid license key."
    category: "Platform"
  - title: "Support for Hibernating an Endpoint"
    description: "Nutanix Enterprise AI now supports endpoint hibernation, which pauses endpoint activity and releases associated compute resources without deleting the endpoint. Hibernated endpoints can be resumed when sufficient compute resources are available."
    category: "Core"
  - title: "Support for Installing with a Self-managed PostgreSQL Database"
    description: "Nutanix Enterprise AI can now be installed on a Kubernetes cluster using a self-hosted or managed PostgreSQL database server, including managed services like Amazon RDS, reducing the need to maintain separate database servers and supporting instance replication for improved disaster recovery."
    category: "Platform"
  - title: "Support for Advanced Configuration of Inference Endpoints"
    description: "Users can now view and configure the maximum number of tokens supported by a Nutanix Enterprise AI instance based on the selected Hugging Face model, CPU or GPU type, and number of GPUs, helping optimize performance by aligning token limits with available system resources."
    category: "MLOps"
  - title: "Support for New Models"
    description: "Nutanix Enterprise AI adds support for new AI LLMs including Hugging Face models (Llama-3.2-11B-Vision-Instruct, Llama-3.2-90B-Vision-Instruct, Llama-4-Scout-17B-16E-Instruct, Llama-Guard-3-8B, Llama-3.3-70B-Instruct-bnb-4bit, Granite-Embedding-107m-Multilingual) and NVIDIA NIMs (Llama-3.3-Nemotron-Super-49b-v1, Llama-3.1-Nemoguard-8b-Topic-Control, Llama-3.1-Swallow-8b-Instruct-v0, Phi-3-Mini-4k-Instruct, Mistral-7b-Instruct-v0)."
    category: "MLOps"
